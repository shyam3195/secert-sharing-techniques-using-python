## Secret sharing techniques using python

### FRONTEND

build in Express node app

#### Dependencies:
**Node**
#### How to Run
Move to Frontend folder
1. Install libraries
`npm install`
2. Start the server
`node index.js`



### BACKEND

Flask Application

#### Dependencies:
**Python**
#### How to Run
Move to Backend folder
1. Install requirements
`pip install -r requirements.txt`
2. Start the server
`flask run`
